# RCSVITCloud

A custom image and file storage platform built for the RCSVIT club.

- 🖼️ **Storage** — Cloudinary (free tier)
- 🗄️ **Database** — Cloudflare D1 (free tier)
- 🚀 **Hosting** — Render (free tier)
